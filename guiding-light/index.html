Introduction (about)

Guiding Light is a very original Discord bot. Its purpose is to manage JTC (Join-To-Create) voice channels.
This means that when you connect to one a temporary is created and you are moved to it. It also has an exclusive feature:
notification channels. When a temporary channel is created for someone, a message about it is sent to a notification channel.
Another feature is the /type command. It allows anyone who joined our community guild to send a message from the bot.
It's not that useful but you can send animated and some of the external emoji thanks to this!

NOTE: Users cannot use emoji of unknown guilds. This means that if the emoji you want to use is in guild A while the bot isn't, instead of emoji you'll see part of its representation, e. g. :emoji_name:
